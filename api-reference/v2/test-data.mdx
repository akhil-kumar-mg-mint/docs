## Test UPI VPA

Use the following VPAs to test UPI transactions.

| VPA | Description | Result |
| :--- | :--- | :--- |
| `success@upi` | Simulates a successful payment | **Success** |
| `failure@upi` | Simulates a failed payment | **Failure** |
| `pending@upi` | Simulates a pending payment | **Pending** |

<Tip>
  When using the `intent` flow on mobile, selecting any UPI app in the simulator will trigger the respective flow based on the test VPA used.
</Tip>
